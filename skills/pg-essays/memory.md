# pg-essays — memory

One-sentence learnings from past conversations. Read this before answering;
append to it when a chat teaches something durable about *how to use this skill
well* (which essays answer which questions, recurring user preferences, common
mistakes to avoid).

**Rules for entries**
- One sentence each. Specific and reusable, not a transcript of a single chat.
- About the *skill's behavior or the corpus*, never about a particular user's
  private details.
- If a new learning contradicts an old one, replace the old line rather than
  stacking both.
- Keep this short. If it grows past ~20 lines, merge or drop the weakest entries.
